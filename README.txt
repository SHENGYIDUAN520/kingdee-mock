星云工作台（静态 HTML 演示）

1. 浏览器打开 index.html，或 python -m http.server 后访问
2. 顶栏选「组织」可切换演示数据种子；黄条提示可关（本机记住）
3. 侧栏 ★ 收藏模块（最多 8 个），收藏分组在顶部
4. 工作台顶部快捷卡片跳转销售/采购/库存/财务
5. 表格：「列显示」勾选可见列（每表单独记忆）；表头排序；表区可滚动且表头固定；筛选、分页、多选与批量审核
6. 地址栏 #/sales、#/inventory 等可直达模块；导出 CSV 与当前可见列一致
7. Ctrl+K 面板无输入时顶部为「最近访问」；偏好里可设表格紧凑/舒适密度
8. 顶栏：通知铃铛、Ctrl+K 命令面板、Ctrl+/ 帮助、主题切换
9. 偏好页可下载操作日志 .log；动态侧栏「复制」复制全部操作记录

数据均为模拟，不连接真实系统。
