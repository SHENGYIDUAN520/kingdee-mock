# 金蝶云·星空 - 企业管理系统

## 项目介绍

这是一个基于HTML5、CSS3和JavaScript开发的企业管理系统前端模拟页面，模仿了金蝶云·星空系统的现代化界面设计。该项目旨在展示企业管理系统的用户界面和基本功能，包括库存管理、客户管理、销售管理、采购管理等模块。

## 功能特点

### 核心功能
- **多模块管理**：支持首页、客户管理、销售管理、采购管理、库存管理等多个功能模块
- **快速操作**：提供其他入库、其他出库、移仓单、调拨出库、调拨入库、调拨差异处理等快速操作
- **待办事项**：显示待审核和未完成的任务
- **实时数据**：展示采购入库、销售出库、销售退货、新增客户等实时业务数据
- **功能列表**：根据选择的模块动态显示相关功能
- **操作日志**：记录用户操作历史和模拟执行记录

### 技术特点
- **现代化界面**：采用深色侧边栏、卡片式布局、渐变色彩等现代设计元素
- **响应式设计**：适配桌面端、平板端和手机端
- **交互式体验**：平滑的动画效果、实时反馈、直观的操作界面
- **模块化结构**：代码组织清晰，易于维护和扩展
- **无障碍访问**：添加了必要的ARIA属性，支持屏幕阅读器

## 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **样式设计**：使用CSS变量实现主题管理，响应式布局
- **交互逻辑**：原生JavaScript实现事件处理和DOM操作
- **数据模拟**：内置模拟数据，无需后端服务
- **响应式设计**：使用媒体查询实现多端适配

## 快速开始

### 本地运行

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python 3
   python -m http.server 8000
   
   # 或使用Python 2
   python -m SimpleHTTPServer 8000
   
   # 或使用Node.js
   npx http-server -p 8000
   ```

3. **访问项目**
   打开浏览器，访问 `http://localhost:8000`

## 部署指南

### GitHub Pages部署

1. **创建GitHub仓库**
   在GitHub上创建一个新的仓库

2. **推送代码**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <repository-url>
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库设置
   - 找到"Pages"选项
   - 选择"main"分支作为源
   - 点击"Save"按钮
   - 等待部署完成，访问生成的GitHub Pages URL

### 其他部署方式

- **静态网站托管**：可部署到Vercel、Netlify、AWS S3等静态网站托管服务
- **服务器部署**：可部署到任何支持静态文件的Web服务器

## 移动端适配

该项目已实现完整的移动端适配，支持以下设备：

- **桌面端**（≥1024px）：完整显示所有功能，侧边栏常驻
- **平板端**（768px-1023px）：侧边栏可折叠，卡片布局调整为2列
- **手机端**（≤480px）：侧边栏通过菜单按钮打开，卡片布局调整为1列

### 移动端操作说明

1. **打开/关闭侧边栏**：点击顶部的菜单按钮（☰）
2. **导航**：点击侧边栏中的菜单项切换模块
3. **快速操作**：点击快速操作卡片发起相应业务流程
4. **待办事项**：点击待办卡片标记任务完成/未完成
5. **实时数据**：点击数据卡片刷新数据或查看详情
