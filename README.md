# 星云工作台 · 企业管理模拟

纯前端静态演示项目：深色/浅色主题、侧边导航、KPI 沙盘、明细表、操作动态与**命令面板**（`Ctrl+K` / `⌘K`）。数据由当日种子随机生成，**不连接任何真实 ERP**。

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript（无构建、无框架）
- 字体：[DM Sans](https://fonts.google.com/specimen/DM+Sans)、[JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)（Google Fonts）

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 入口，挂载 `#app` 与 Toast 容器 |
| `kingdee-mock.css` | 主题变量、布局、命令面板与响应式 |
| `kingdee-mock.js` | 路由状态、模拟数据、本地存储、交互 |

## 本地运行

```bash
cd kingdee-mock
python -m http.server 8000
```

浏览器打开 `http://localhost:8000`（直接双击打开 `index.html` 也可，部分浏览器对 `file://` 限制本地存储）。

## 功能一览

- **组织**：顶栏下拉切换「总部 / 华东 / 华南」，会写入本地并**改变 KPI 与表格的随机种子**（演示用）
- **预警条**：首次进入显示黄色提示条，可关闭；关闭状态会记住（偏好里「重置」可恢复）
- **通知**：顶栏铃铛，演示消息列表；点开某条即标为已读并更新角标
- **收藏**：侧栏每条导航右侧 ★，最多 8 个；收藏项置顶在「收藏」分组
- **工作台快捷入口**：工作台顶部卡片一键跳转销售 / 采购 / 库存 / 财务
- **表格**：多选（含本页全选）、**批量审核**、**列显示**（按数据表独立隐藏列，本地保存）、**表头点击排序**、「清除排序」、筛选、分页（每页 8 条）；表体区域可纵向滚动且**表头吸顶**；**点击行**打开详情抽屉；**导出 CSV** 与当前排序及**可见列**一致
- **地址栏路由**：使用 `#/模块id` 直达页面（如 `https://…/index.html#/inventory`），切换模块时地址同步更新（`replaceState`）
- **命令面板**：无关键字时列表顶部为 **「最近访问」**（最多 5 条，随导航更新）
- **偏好**：**表格密度**「舒适 / 紧凑」影响业务表明细行高与字号
- **详情抽屉**：业务字段 + **四步模拟审批流**（保存 → 部门 → 财务 → 过账），「下一步」为纯前端演示
- **导出**：各业务页「导出表格 CSV」下载当前筛选结果；**偏好**页「下载 .log」导出操作日志
- **动态**：侧栏「复制」将操作记录复制到剪贴板

## 快捷键与操作

- **Ctrl+K**（macOS：**⌘K**）：打开/关闭命令面板，快速跳转模块
- **Ctrl+/**（macOS：**⌘/**）：打开快捷键说明
- **Esc**：关闭命令面板、详情抽屉、帮助窗口或通知浮层（焦点在页面时）
- 顶栏可切换**浅色 / 深色**；**偏好**页可重置本地状态
- 「模拟审核 / 模拟同步」会写入右侧动态与本地日志（最多保留 80 条）
